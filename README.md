
# Spotify

Spotify Clone application built with Spotify API, Expo Router, Expo-AV(Audio)


## Features

- Custom music player
- Music list displays with Spotify style
- Adding favorite music and displaying
- Cross platform
- Music playing with expo audio
- Neon DB for database
